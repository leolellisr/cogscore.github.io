CogScore - An Online Evaluation Playground For Cognitive Architectures With Developmental Robotics.

This work aims to implement a general framework for the automated evaluation of different levels of diverse cognitive capabilities of cognitive architectures. The architectures will be implemented with CST, the Cognitive Systems Toolkit.
